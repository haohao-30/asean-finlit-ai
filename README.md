# 🌏 ASEAN FinLit AI

ASEAN Financial Literacy & Risk Protection System

## 🚀 Quick Start

```bash
# 1. Clone repository
git clone https://github.com/haohao-30/asean-finlit-ai.git
cd asean-finlit-ai

# 2. Create virtual environment
python -m venv .venv
.venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Create .streamlit/secrets.toml with your Gemini API key
# GEMINI_API_KEY = "your-api-key-here"

# 5. Run app
streamlit run xixixi.py
